# bc.hlx - Xposed 模块

## 模块介绍

本模块为葫芦侠/葫芦侠3楼提供增强功能

- 自动全版块签到（每日一次）
- 免费主题必选项
- 屏蔽主题付费弹窗
- 显示部分已删评论
- 为已删除评论添加【已删除】标签
- 跳过启动页
- 禁用版本更新

## 小号 Key

用于点赞、刷新浏览量状态。

## 浏览量控制

当前可用请求次数：**500**（长按数字可修改）

## 快捷操作

- 🔄 一键刷新收藏帖浏览量
- 🎁 一键送最新收藏帖葫芦

## 批量控制

- ⚡ 一键禁用所有
- 🔘 关闭
- 🔄 一键启用所有
